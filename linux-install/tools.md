# 装机必备工具

## Windows U盘刻录工具

- [Rufus](https://rufus.ie/zh/):一个轻量、快速且兼容性强的工具，用于将 ISO 写入 U 盘制作可启动盘
- [Ventoy](https://www.ventoy.net/cn/download.html):安装一次后可直接复制多个 ISO 文件到 U 盘，实现多系统启动和免重复格式化
- [balenaEtcher](https://etcher.balena.io/#download-etcher):跨平台、操作简单的工具，专注于快速写入 ISO/IMG 文件到 U 盘或 SD 卡
- [UltraISO](https://www.ultraiso.com/download.html):功能全面的 ISO 管理软件，可以编辑、转换和写入 ISO 镜像到 U 盘
- [PowerISO](https://www.poweriso.com/download.php):一体化 ISO 工具，支持创建、编辑、压缩和写入 ISO 到 U 盘，兼容多种光盘镜像格式
