\subtitle{平行时空的我们}

\subsubtitle{你的时空}

\large
\begin{spacing}{2}
    在那个平行的时空里，你正漫步在吉职的林荫道上。
    那时的你，还不知道在北京的校园里，有一个男孩会在未来成为你生命中最重要的人。
\end{spacing}

\subsubtitle{我的时空}

\large
\begin{spacing}{2}
    与此同时，在北京的学院路，我正在中国矿业大学的校园里过着截然不同的生活。
    那时的我，从未想到命运会在某个时刻让我们相遇。
\end{spacing}

\subsubtitle{时空交汇}

\large
\begin{spacing}{2}
    我们就像两颗各自运行的星星,你在南方的暖阳里细嗅花香;我在北国的风雪中静听雪落;
    你在小镇上感受人间烟火;我在都市里瞥见车水马龙;吉安与北京近二千公里的距离,却阻隔不了两颗注定要相遇的心。
\end{spacing}

\bottomtext{“人生就像两条平行线，在某个特定的时刻突然交汇，然后紧紧缠绕在一起。”}

\subtitle{相遇——初见几何}

\large
\begin{spacing}{2}
    我记得再清楚不过了，那是2023年的冬天，空气里已经能呵出白气的季节。
    世界好像被罩在一个巨大的、冰冷的玻璃罩子里。

    那天晚上，我结束了一天的学习，瘫在椅子里漫无目的地刷着手机。
    就在某个学习群聊的角落里，你的头像突然跳进视线——是个衣着暴露的擦边女头像，配上奇怪的群昵称。
    我手指一顿，脑子里瞬间闪过“卖片的”。你说你数学不好，需要有人辅导你学习数学，一开始找上了陆总，
    陆总就不约而同地把你拉进了群聊。

    鬼使神差地，我点开你的头像，发送了好友申请，你出乎意料的居然秒同意了好友申请。
    我开始质问你是不是卖片的，你义正言辞地说你不是，并且开始介绍起自己，
    就这样，我们的话匣子打开了。从生活的趣事，聊到最近的电影，从吐槽冬天冻手冻脚，
    到分享各自喜欢的电影风格，再到生活的各种琐事，总之聊的热火朝天。

    那个寒冷的冬夜，因为手机屏幕另一端你的存在，变得无比温热。
    我的生活原来很无聊，但我觉得有你的出现，整个世界都亮了。
    我们好像有说不完的话。从深夜聊到凌晨，互道“晚安”后，
    又忍不住分享一条有趣的视频、彼此心照不宣的寒暄，那一刻，
    我看着屏幕上那几行简单的字，心脏像被温暖的潮水包围。
    窗外也许还是灰蒙蒙的冷，但我的胸口却烫得像揣了一个小太阳。

    从那一刻起，我的夜晚，不再是一个人的独白。 
    你的存在，通过那小小的屏幕，穿越了寒冷的城市，永远地、坚定地，站在了我身边。  
    
    \bottomtext{“你的出现，让我的世界有了新的维度”}
\end{spacing}

\vspace*{1cm}

\large
\begin{spacing}{2}

        说不清具体是哪一刻心动，\\

        也许是你认真说话时的侧脸。\\[0.3cm]

        在不知不觉中发芽。\\

        心里已经开满了花。\\[0.3cm]
        
        原来喜欢一个人，\\
        不需要什么惊天动地的理由，\\
        只是在某个平凡的瞬间，\\
        突然觉得，就是你了。
    \end{center}
    
\end{spacing}
