# Ubuntu 系统安装ROS2

注意不同的Ubuntu发行版对应不同的ros2版本，具体的版本可以通过以下命令进行查看

```bash
cat /etc/os-release
```

![查看Ubuntu版本号](https://raw.githubusercontent.com/doublenuo/ubuntu-image/main/image/20260324092654853.png)

具体Ubuntu与ROS的版本对应可以参考以下这个表

| Ubuntu版本 |  ROS2版本   |
| :--------: | :---------: |
| Ubuntu 20  |  ROS2 foxy  |
| Ubuntu 22  | ROS2 humble |
| Ubuntu 24  | ROS2 jazzy  |

## Ubuntu 20安装ROS2 foxy

### 修改国内apt镜像源

安装过程中需要安装一些第三方软件，使用默认的Ubuntu软件源可能会有网络问题（有梯子的同学可以忽略这个问题）

#### chsrc自动换源

这里推荐[chsrc](https://github.com/RubyMetric/chsrc)这个全平台通用换源工具，安装方法为

```bash
wget https://gitee.com/RubyMetric/chsrc/releases/download/pre/chsrc_latest-1_amd64.deb
sudo dpkg -i chsrc_latest-1_amd64.deb
```

有了这个工具之后就可以简单的换源了，这个工具会自动测速选择当前网络下最快的软件源，并自动更新软件源

```bash
sudo chsrc set ubuntu
```

![chsrc进行自动换源](https://raw.githubusercontent.com/doublenuo/ubuntu-image/main/image/20260324090821634.png)

#### 手动换源

如果不信任第三方工具的话，也可以手动维护并更换软件源，首先需要备份好原来的软件源,然后修改原来软件源的配置

```bash
sudo cp /etc/apt/sources.list.d/ubuntu.sources /etc/apt/sources.list.d/ubuntu.sources.backup
sudo vim /etc/apt/sources.list.d/ubuntu.sources
# 将内容全部删除后替换为清华软件源
# ubuntu 20设置为以下内容
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ focal main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ focal-updates main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ focal-backports main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ focal-security main restricted universe multiverse
# ubuntu 22设置为以下内容
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-updates main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-backports main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-security main restricted universe multiverse
# ubuntu 24设置为以下内容
Types: deb
URIs: https://mirrors.tuna.tsinghua.edu.cn/ubuntu/
Suites: noble noble-updates noble-backports noble-security
Components: main restricted universe multiverse
Signed-By: /usr/share/keyrings/ubuntu-archive-keyring.gpg
```

修改完记得更新一下软件源

```bash
sudo apt update
```

### 前置配置

#### 设置字符编码

ROS2 强依赖 UTF-8 locale，非UTF-8编码可能会出现一些意料不到的错误，ubuntu系统默认使用UTF-8编码，可以使用以下命令进行查看。

```bash
locale
# 若提示没有这个命令，可进行安装
sudo apt install locales -y
# 若发现不是UTF-8编码，可以进行以下修改
sudo locale-gen en_US en_US.UTF-8
sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8
export LANG=en_US.UTF-8
```

![locale查看字符编码](https://raw.githubusercontent.com/doublenuo/ubuntu-image/main/image/20260324093125674.png)

#### 安装基础依赖

```bash
sudo apt install software-properties-common curl -y
sudo add-apt-repository universe
```

### 开始安装

#### 添加GPG Key

ROS2 软件包是签名的，为了让系统信任 ROS2 源，需要添加官方 GPG Key。

```bash
sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key -o /usr/share/keyrings/ros-archive-keyring.gpg
```

-   `-o /usr/share/keyrings/ros-archive-keyring.gpg`: 将 key 保存到系统 keyrings 目录
-   如果网络访问github比较慢，可以选择[ghproxy](https://gh-proxy.com/)镜像

#### 添加ROS2软件仓库

使用国内镜像可以大幅提高下载速度。

```bash
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] \
https://mirrors.tuna.tsinghua.edu.cn/ros2/ubuntu $(. /etc/os-release && echo $UBUNTU_CODENAME) main" \
| sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null

sudo apt update # 记得更新一下软件缓存
```

#### 安装ROS2 foxy

开发推荐安装ROS2 foxy桌面版，包括ROS2核心库、Rviz2 可视化工具、rqt工具等等

```bash
sudo apt install ros-foxy-desktop python3-argcomplete
```

若是服务器或者docker环境，可以只安装最小基础版，仅仅包括ROS2核心功能，不包括GUI

```bash
sudo apt install ros-foxy-ros-base python3-argcomplete
```

#### 安装开发工具

`ros-dev-tools`是 **ROS 官方提供的一组辅助开发工具包**，主要是为了 **加速开发、调试和管理 ROS2 包**，其包括了OS2 官方推荐的构建系统`colcon`等工具

```bash
sudo apt install ros-dev-tools
```

#### 配置环境变量

为了每次打开终端自动加载 ROS2 环境，可以将ROS2写入环境变量	

```bash
echo "source /opt/ros/foxy/setup.bash" >> ~/.bashrc
source ~/.bashrc
```

#### ROS2第一个节点

```bash
ros2 run turtlesim turtlesim_node
```

![ROS2 foxy第一个节点](https://raw.githubusercontent.com/doublenuo/ubuntu-image/main/image/20260324094036816.png)

## 卸载ROS2

```bash
# 卸载ROS2 foxy
sudo apt remove '~nros-foxy-*'' && sudo apt autoremove
# 卸载 ROS2 humble
sudo apt remove '~nros-humble-*' && sudo apt autoremove 
# 卸载 ROS2 jazzy
sudo apt remove '~nros-jazzy-*' && sudo apt autoremove 

# 如果需要卸载软件仓库，可以执行以下命令
sudo apt remove ros2-apt-source
sudo apt update
sudo apt autoremove
```

## Gazebo安装

Gazebo也要与当前安装的ROS版本相匹配，注意ROS2 foxy已停止维护，因此可能需要其他手段安装Gazebo，其他版本的ROS2对于的Gazebo可以看下图。

![Gazobo与ROS2版本对应](https://raw.githubusercontent.com/doublenuo/ubuntu-image/main/image/20260324100456702.png)

### 安装ROS2与Gazebo的桥接包

`ros_gz` 是 ROS2 和 Gazebo整合桥接包,它提供 **消息桥接、控制接口、插件接口**，用于机器人仿真

```bash
sudo apt-get install ros-${ROS_DISTRO}-ros-gz
```

### 安装Gazebo

首先需要添加Gazebo软件仓库

```bash
sudo curl -sSL https://packages.osrfoundation.org/gazebo.key | sudo gpg --dearmor -o /usr/share/keyrings/gazebo-archive-keyring.gpg
sudo tee /etc/apt/sources.list.d/gazebo.list > /dev/null <<EOF
deb [signed-by=/usr/share/keyrings/gazebo-archive-keyring.gpg] https://packages.osrfoundation.org/gazebo/ubuntu $(lsb_release -cs) main
EOF

# 添加完软件仓库记得更新一下缓存
sudo apt update
```

开始安装Gazebo,这里需要看清楚你需要安装的版本

```bash
sudo apt install gz-harmonic # 替换为你对应的版本
```

测试是否安装完成

```bash
gz sim
```
