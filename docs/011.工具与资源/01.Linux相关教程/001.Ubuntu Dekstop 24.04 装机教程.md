# Ubuntu Dekstop 24.04 装机教程

## 磁盘分区配置

首先需要准备一块空白且未建立分区的磁盘，并在交互安装过程里选择手动配置分区，按照以下方案进行分区。
1. boot分区:512MB-1GB之间即可,文件系统为ext4,挂载点为/boot,这个分区存放内核、GRUB启动
2. swap分区：swap分区的作用是虚拟内存，但对于现代16GB的内存而言几乎不需要使用swap分区,不过可以提升系统稳定性，小于16GB的内存建议swap分区大小为内存的两倍，大于16GB的内存swap分区大小为16GB即可；
3. HOME分区:挂载点/home,用户目录;
4. 根目录分区:挂载点/,系统目录。

例如一个16GB内存、500GB的硬盘可以这样设置分区
```text
/boot: 1GB
swap: 16GB
/: 100GB
/home: 剩余全部空间
```

## 安装中文拼音输入法

建议使用fcitx5,比其他的输入法更稳定
```bash
sudo apt update
sudo apt install fcitx5 fcitx5-chinese-addons fcitx5-configtool
```

执行以下命令并设置 Fcitx 为默认输入法，执行以下命令后必须注销或者重启系统才可以生效

```bash
im-config
```

配置环境变量并设置开机自动启动
```bash
vim ~/.profile
# 以下是需要写入文件的内容
export GTK_IM_MODULE=fcitx
export QT_IM_MODULE=fcitx
export XMODIFIERS=@im=fcitx
# 设置开机自启动
mkdir -p ~/.config/autostart
cp /usr/share/applications/org.fcitx.Fcitx5.desktop ~/.config/autostart/
```
最后需要添加中文拼音输入法,执行交互式安装命令，选择拼音输入法即可
```bash
fcitx5-configtool
```

## 配置gnome桌面插件

首先安装一下必要的软件
```bash
sudo apt update
sudo apt install gnome-tweaks -y 
sudo apt install chrome-gnome-shell -y
sudo apt install gnome-shell-extension -y
sudo apt install gnome-shell-extension-manager -y
```
