-   Data Link: None

### 论文介绍

如何让腿式机器人在未知、极端、非结构化环境中完成类似 parkour 的高难度动态运动。这篇论文的核心思想是通过强化学习学动态可行性（dynamic feasibility），让机器人直接学会何时跳、跳多远、如何落地。
