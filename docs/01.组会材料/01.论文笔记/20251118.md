## Agile But Safe: Learning Collision-Free  High-Speed Legged Locomotion

-   Project Link: [https://agile-but-safe.github.io/](https://agile-but-safe.github.io/)
-   Github Link:[https://github.com/LeCAR-Lab/ABS](https://github.com/LeCAR-Lab/ABS)
-   Paper Link: [https://arxiv.org/abs/2401.17583](https://arxiv.org/abs/2401.17583)
-   Data Link: None
-   Docs Link: None

### 论文介绍

现代强化学习已经能让四足机器人实现高速奔跑、跳跃和敏捷运动，但存在一个问题**速度越快的情况下碰撞风险指数级上升**，因此这篇论文的核心目标是如何让四足机器人同时实现高速且无碰撞。
