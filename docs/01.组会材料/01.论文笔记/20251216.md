## Gallant: Voxel Grid-based Humanoid Locomotion and  Local-navigation across 3D Constrained Terrains

-   Project Link: [https://gallantloco.github.io/](https://gallantloco.github.io/)
-   Github Link: [https://github.com/InternRobotics/Gallant](https://github.com/InternRobotics/Gallant)
-   Paper Link: [https://arxiv.org/abs/2511.14625](https://arxiv.org/abs/2511.14625)
-   Data Link: None
-   Docs Link: None

### 论文介绍

如何让人形机器人在复杂三维受限环境中同时完成能走 + 能规划 + 不碰撞的局部导航与运动控制，这篇论文的核心思想是使用 3D voxel grid 统一表示环境，让强化学习策略直接从三维占据空间学习可通行性和locomotion 控制。
