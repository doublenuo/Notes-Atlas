## VLA-RL: Towards Masterful and General Robotic Manipulation with Scalable Reinforcement Learning

- Project Link: None
- Github Link: [https://github.com/GuanxingLu/vlarl](https://github.com/GuanxingLu/vlarl)
- Data Link: None
- Paper Link: [https://arxiv.org/abs/2505.18719](https://arxiv.org/abs/2505.18719)
- Docs Link: None

### 论文介绍

传统的 VLA 模型通过模仿学习从离线演示数据中学习，无法有效应对未在数据中出现过的状态，这篇论文通过在线 RL 引入探索机制，使模型能够在测试时通过与环境交互不断提升表现；其次在机器人操作任务中，环境通常只在任务完成时给出一个奖励信号，也就是稀疏奖励问题，导致 RL 训练困难，作者提出的 RPRM 模型能够为中间步骤提供密集奖励，有效引导学习过程；最后通过一些工程优化的手段实现在大规模 VLA 上的高效 RL 微调。

![VLA-RL 论文介绍图](https://raw.githubusercontent.com/doublenuo/ubuntu-image/main/image/20260326133305800.png)

### 创新点

- **轨迹级 RL 建模**：将机器人操作任务建模为多模态、多轮对话，使得 RL 可以适配自回归 VLA 的结构；
- **机器人过程奖励模型**：解决稀疏奖励问题；
- **工程优化**：GPU 负载均衡、批量解码、critic warmup 等等

### 模型框架

基础预训练模型选用 OpenVLA-7B，每个时间步 $t$，它将第三人称相机捕获的图像 $o_t$ 和人类指令 $v_t^{\mathrm{in}}$ 作为输入并输出动作令牌序列 $v_t^{\mathrm{out}}$，其中每个动作令牌表示机器人动作空间离散的一维。使用后处理函数 $f$ 从该序列中提取最终机器人动作 $a_t=f(v_t^{\mathrm{out}})$.

![VLA-RL 模型框架图](https://raw.githubusercontent.com/doublenuo/ubuntu-image/main/image/20260326133336939.png)

#### 问题定义

首先将马尔可夫决策过程公式化为多轮对话，空间 $V_m$ 和 $V_n$ 分别代表可能的输入和输出文本序列，其中 $m$ 和 $n$ 指定输入和输出的最大序列长度，状态空间定义为 $S=O\times V_m$，其中 $O$ 表示图像空间，动作空间由 VLA 生成的所有可能的输出话语的集合 $V_n$ 给出，VLA 策略就可以这样表示

$$
\pi_{\theta}: O\times V_m \to V_n
$$

在每个时间步，策略 $\pi_{\theta}$ 是一个概率分布 $\pi_{\theta}({V_t^{\mathrm{out}}|o_t,\theta}(V_t^{\mathrm{in}})$，且使用 PPO 算法进行稳定的训练。

#### 整体流程

首先初始化环境，初始化预训练 VLA 模型的参数 $\theta_0$，初始化过程奖励模型的参数 $\phi$，加载后处理函数 $f$，步长 $M$，折扣因子 $\gamma$，优势函数的参数 $\lambda_{\mathrm{GAE}}$，在 N 个并行的环境中 rollout 收集一下轨迹数据，对于每个时间步

## Diffusion-Based Imaginative Coordination for Bimanual Manipulation

- Project Link: []
- Github Link: []
- Data Link: []
- Paper Link: [https://arxiv.org/abs/2507.11296](https://arxiv.org/abs/2507.11296)
- Docs Link: []

### 论文介绍

双臂机器人通过 skill primitives 具有强任务依赖性，而共享观测导致会双臂仅依赖当前观察无法形成真正协调，这篇论文提出了一个统一的基于扩散模型的框架，用于视频和动作预测的联合优化，同时引入单向注意力机制，视频预测以动作为条件，而动作预测仍然独立于视频预测，这样可以保证在推理时省略视频预测，显著提高效率。

![image-20260329103110751](https://raw.githubusercontent.com/doublenuo/ubuntu-image/main/image/20260329103110778.png)

### 创新点

- **将未来帧预测引入双臂控制**，作为两只机械臂共享的未来想象空间来实现隐式协调，而不是依赖手工规则或角色分配；
- 引入 **统一的 Diffusion 框架** 联合学习动作与未来状态；
- 引入 **单向注意力机制**，训练时视频预测依赖动作学习因果关系，推理时动作生成不依赖视频

### 模型架构图

模型整体主要分为三个核心阶段：**观测标记化与嵌入**、**未来帧的紧凑表示**，以及 **联合去噪扩散解码**

模型接收两类观测：**单目 RGB 图像** 和 **机器人的本体感知状态**，RGB 图像经过一个基于 ResNet-18 的编码器，提取出空间特征图，随后展平并投影为 $H\times W$ 个视觉 token，并添加 2D 位置编码以保留空间结构。与此同时，本体感知状态通过一个 MLP 映射到相同的隐空间维度。这两类 token 被拼接，送入右侧的 **Transformer 编码器**，生成最终的 **观测嵌入**。这个嵌入将作为后续扩散模型的条件信号。

接下来是关键的未来帧处理。为了高效建模，我们并没有直接预测像素，而是将未来帧输入一个 **预训练的视频分词器**，将其压缩为紧凑的潜在视频 token。再经过分块和时空位置编码，形成结构化的视频表示。这样做可以大幅降低计算冗余，同时保留高层的时序动态信息。

最后看最核心的 **联合去噪解码器**，在训练阶段，我们同时向 **动作序列** 和 **视频 token** 注入噪声，得到一个共享的噪声潜变量。解码器基于扩散步编码和之前得到的观测嵌入，通过 **交叉注意力机制** 进行迭代去噪。这里采用了 **单向注意力机制**：动作 token 之间互相可见，但视频 token 可以同时关注动作 token 和自身。这种设计使得视频预测能够学习到动作对未来画面的因果关系。解码后的特征分别送入两个预测头：一个线性层用于重建动作序列，一个 MLP 用于重建视频 token。

在推理阶段，得益于单向注意力机制，动作生成不再依赖视频 token，因此我们可以 **完全跳过视频预测分支**，只运行动作去噪过程，从而显著降低推理延迟，满足高频控制需求。

![image-20260329103056549](https://raw.githubusercontent.com/doublenuo/ubuntu-image/main/image/20260329103056590.png)

#### 问题定义

作者将双手操作公式化为条件生成问题，并提出了一个统一的基于变换的扩散模型，用于动作预测和视频预测。由于视频预测和动作预测之间存在内在联系，因此该双臂问题被定义为在一个统一的生成框架下，给定当前的相机观测值和机器人本体感知 $(o_t,p_t)$，模型会输出未来 $N$ 帧的动作和观测值

$$
\pi(o_t, p_t)\to (a_t: a_{t+N}, o_{t+1}, o_{t+1+N})
$$

#### 输入 token 化

观察由 **视觉信息** 和 **机器人本体感受状态** 组成，单视图 RGB 图像通过基于 ResNet-18 的 $2D$ 图像编码器来提取空间特征，产生 $2\mathrm{D}$ 特征图。然后将特征图平坦化并投影成 $H\times W$ 视觉标记，每一个都表示为一个 $\mathrm{d}$ 维向量。一个 $2\mathrm{D}$ 位置编码被添加到每个视觉标记中以保持空间结构。类似地，机器人的本体感受状态通 MLP 被映射到相同的潜在空间。最终的观察表示通过将视觉标记与本体感受嵌入连接起来获得，这些 token 由由多个堆叠层组成的 Transformer 编码器处理，以构造观测嵌入 $e$，用于扩散模型的条件输入。

#### 视频预测

为了实现有效的视频预测，作者执行均匀的时间采样以确保帧间有足够的视觉变化，然后使用预训练的 Cosmos 视频标记器对采样帧进行编码。视频标记器可以减少时间冗余并显着降低计算复杂度，同时保留足够的信息用于视频重建。与 ViT 类似，我们对获得的潜在特征应用补丁化过程并将其划分为非重叠补丁，每个未来的令牌进一步用时空位置嵌入来丰富。这种丰富的令牌为未来帧提供了结构化和计算高效的表示，从而允许对高级时间动态进行更有效的建模。

#### 条件扩散模型

给定先前获得的观察嵌入 e 和扩散步骤 k，我们采用条件扩散模型来迭代地细化未来动作和视觉标记的噪声表示。在每个扩散步骤中，噪声被注入到干净的动作序列 $a$ 和未来的帧标记 $v$ 中。这些噪声表示然后经由线性层被投影到潜在空间中。这些潜在嵌入使用交叉注意从观察嵌入和扩散步骤编码中提取信息。

#### 单向注意力机制

作者在去噪解码器中的交叉注意模块的自我注意层内设计了一个单向注意机制，以调节动作和未来帧令牌之间的信息流。**动作令牌只关注自己，而未来帧令牌关注自己和历史动作令牌**，使模型能够捕获所执行的动作和它们相应的视觉结果之间的时间依赖性。这种设计提供了两个关键的优点：

1. 首先，在训练期间，允许未来帧标记关注过去动作标记使模型能够捕获动作和未来视觉状态之间的因果关系，促进了动态环境的学习。
2. 其次，在推理期间，动作生成独立于未来的视觉令牌，消除了对视频预测的需要。这减少了计算开销，降低了推理延迟，并提高了控制频率，这对于实时机器人执行至关重要。

![单向注意力机制](https://raw.githubusercontent.com/doublenuo/ubuntu-image/main/image/20260330172740106.png)

#### 推理与执行

在推理期间，省略视频预测，并且仅执行动作生成以减少延迟并提高控制频率。这是通过单向注意机制实现的，其中动作令牌不接收来自未来视觉令牌的信息。给定从高斯噪声采样的初始动作序列 $a^{k}\sim N(0,1)$，该模型通过反向扩散过程迭代地细化动作令牌。动作块以开环方式执行，直到完成，之后生成下一个动作块。

### 结论与展望

本文提出了一种统一的基于扩散的视频预测和动作拟合联合框架，以端到端的方式优化感知-预测-控制过程，通过多帧潜在预测策略和单向注意机制，方法有效地利用预测预见来增强双手协调。实验结果表明，在模拟和真实的之间，虽然方法显著提高了协调和控制效率，但未来的工作可以探索将多模态感觉反馈和自适应规划结合起来，以增强动态和非结构化环境中的鲁棒性。

虽然方法表现出了很强的性能，但仍有几个方向是开放的。它可能会受益于更强大的视觉编码器来处理涉及小物体或遮挡物体的细粒度任务松散耦合任务中的轻微下降表明未来将探索特定于手臂的表示，以避免不必要的依赖性。选择信息量更大的关键帧作为预测目标而不是均匀采样可以增强时间推理和协调。
